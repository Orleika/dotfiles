# dotfiles
Orleika's dotfiles

## Installation
```
$ git clone https://github.com/Orleika/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ exec ./setup.sh
```

## Requirements
* git
* zsh
* [Pygments](http://pygments.org/)
  * `pip install Pygmetns`
